AutoPrime Garage

Aplicativo mobile desenvolvido em React Native com TypeScript para gerenciamento de veículos.

Sobre o Projeto

O AutoPrime Garage foi desenvolvido como trabalho da disciplina de Desenvolvimento para Dispositivos Móveis. O aplicativo permite cadastrar, visualizar, editar, reservar e excluir veículos utilizando integração com banco de dados em nuvem.

O projeto foi desenvolvido utilizando React Native com Expo e TypeScript, além de integração com Supabase para persistência dos dados.

Tecnologias Utilizadas

- React Native
- Expo
- TypeScript
- Supabase
- React Navigation
- GitHub Pages

Funcionalidades

- Cadastro de veículos
- Listagem de veículos
- Visualização de detalhes
- Edição de veículos
- Reserva de veículos
- Exclusão de veículos
- Tema escuro/claro
- Navegação entre telas
- Persistência em banco de dados online

Instalação e Execução

Clone o repositório:

git clone https://github.com/lukaszwx/autoprime-garage.git

Acesse a pasta do projeto:

cd autoprime-garage/autoprime

Instale as dependências:

npm install

Execute o projeto:

npm run web

Deploy

Publicação realizada com GitHub Pages.

Executar:

npm run deploy

URL do projeto:

https://lukaszwx.github.io/autoprime-garage/

Banco de Dados

O projeto utiliza Supabase como banco de dados em nuvem para armazenamento e persistência das informações dos veículos.
