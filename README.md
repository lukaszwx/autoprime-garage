# AutoPrime Garage

Projeto Expo React Native com TypeScript.

## Rodar localmente

```bash
npm install
npm run web
```

## Deploy no GitHub Pages

Repositorio:
`https://github.com/lukaszwx/autoprime-garage`

URL esperada:
`https://lukaszwx.github.io/autoprime-garage`

Comandos:

```bash
npm install
npm install --save-dev gh-pages
npm run deploy
```

O script `predeploy` exporta a versao web para a pasta `dist`.
O script `deploy` publica a pasta `dist` no GitHub Pages.
